Understand data by loading, inspecting, and visualizing it.

### Dataset:

Iris Dataset

### Tools Used:

* Pandas
* Matplotlib
* Seaborn

* Created scatter plots, histograms, and boxplots

### Results:

Identified patterns, feature relationships, and outliers through visualization.



##  Task 2: Predict Future Stock Prices

### Objective:

### Dataset:

Yahoo Finance (using yfinance)

### Model Used:

* Linear Regression

### Key Steps:

* Retrieved stock data (Open, High, Low, Volume)
* Trained regression model
* Compared predicted vs actual prices

### Results:

##  Task 3: Heart Disease Prediction

### Objective:

Predict whether a person is at risk of heart disease.

### Dataset:

Heart Disease UCI Dataset

### Model Used:

* Logistic Regression

### Key Steps:

* Data cleaning and preprocessing
* Converted target into binary
* Feature scaling and encoding
* Model training

### Evaluation:

* Accuracy
* Confusion Matrix
* ROC-AUC Curve

### Results:

Model achieved good accuracy. Important features include age, cholesterol, and blood pressure.
